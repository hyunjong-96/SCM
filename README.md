# SCM

## 프로젝트 목적
- - -
Github의 OpenAPI를 이용해서 나의 github를 관리하는 웹 서비스

## 개발 목적
- - -
1. SubModule
   1. 서버 모듈과 화면모듈
2. Spring Security
   1. 구조 파악
   2. Oauth2 사용
3. OpenAPI
4. 아키텍처
   1. 다양한 아키텍처 고민해보기
5. 이론 실습
   1. Spring 3대요소
   2. Filter, Interceptor
   3. 자바함수 - stream
6. 기록
   1. 설정부터 완성까지 기록하기
7. 화면 개발
    1. Vue.js
